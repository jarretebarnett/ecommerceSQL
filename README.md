##### SCRIPTS ⤬ DEPENDENCIES
* ###### SCRIPTS
    * ###### `npm install` ⤬ Command to install `package.json` dependencies locally.
    * ###### `node seeds/index.js` ⤬ Command to seed database locally.
    * ###### `node server.js` ⤬ Command to connect to database using a localhost.
